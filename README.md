# Практическая работа № 3. Разметка веб‑страницы с использованием HTML5

## Краткое описание проекта

Проект демонстрирует применение семантической разметки HTML5, включая:
- структуру страницы с `<header>`, `<nav>`, `<main>`, `<article>`, `<aside>`, `<footer>`;
- форму с валидацией и accessibility‑атрибутами;
- таблицу данных с семантическими элементами и доступной разметкой.

## Инструкции по открытию

1. Распакуйте архив в любую папку.
2. Откройте файл `index.html` в браузере (двойной клик или через контекстное меню → «Открыть с помощью…»).
3. Для корректного отображения стилей убедитесь, что файл `css/style.css` находится в правильной папке.

## Используемые браузеры

Проект протестирован в:
- Google Chrome (последняя версия)
- Mozilla Firefox (последняя версия)
- Microsoft Edge (последняя версия)

## Особенности реализации

- **Accessibility**: ARIA‑атрибуты (`role`, `aria-label`), семантические теги, подписи к полям формы.
- **Валидация**: атрибуты `required`, `pattern`, `minlength`.
- **Адаптивность**: медиа‑запросы для мобильных устройств.
- **Стили**: полосатые строки таблицы, эффекты наведения, единообразный дизайн форм.
