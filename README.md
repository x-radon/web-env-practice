# Настройка рабочего окружения

## Установка ПО

1. **VS Code**: скачать .deb-пакет и установить через `sudo apt install`.
2. **Node.js**: установить через репозиторий NodeSource.
3. **Git**: `sudo apt install git`.

## Запуск проекта

1. Через Live Server в VS Code: клик правой кнопкой по index.html → Open with Live Server.
2. Через Node.js:
   - `npm init -y`
   - `npm install --save-dev http-server`
   - `npm run start` (сервер на http://localhost:8080).

## Версии ПО

- Node.js: v18.20.4
- npm: 10.11.0
- Git: 2.47.0
